# ThinkQuiz

AI-powered gamified learning platform that helps developers learn through AI-generated quizzes, hints, and guided problem solving.

## Features

- LeetCode Quiz Mode
- Concept Learning Mode
- AI Generated Hints
- XP & Levels
- Daily Streaks
- Leaderboards
- Progress Tracking

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Shadcn/UI

### Backend
- FastAPI
- Gemini AI

### Database
- PostgreSQL (Supabase)

### Cache
- Redis (Upstash)
